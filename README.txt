该目录下文件用于学习C++算法
一、容器学习
container_list.cpp : 学习cpp列表容器
container_map.cpp : 学习cpp字典容器
container_priorityQueue.cpp : 学习cpp优先队列容器
container_queue.cpp : 学习cpp队列容器
container_vector.cpp : 学习cpp向量容器

二、数据结构
HuffmanTree.cpp : 学习哈夫曼树的构建
LinkList.cpp : 学习cpp链表

三、计算方法
GCM_LCM.cpp : 学习如何计算最大公约数和最小公倍数
paren_matching.cpp : 基于栈的括号匹配
prime.cpp ： 关于素数的算法
simpleCulculator.cpp ：简单计算器的实现
sortingCollection.cpp ：经典排序算法合集
systemTransformation.cpp : 进制转换
stringProcessing.cpp : 字符串处理
fib.cpp : 斐波那契数列常见思路
coinChange.cpp : 凑零钱问题,经典动态规划问题
best_substruct.cpp : 最优子结构详解
bin_tree_traversal.cpp : 各种二叉树的遍历方法和应用
